FG World Cup AI V5 - Official Squads

Questa versione NON usa più rose inventate.
Ha un import automatico dal PDF ufficiale FIFA SquadLists-English.pdf.

Deploy Render:
- Build Command: npm install
- Start Command: npm start

Dopo il deploy:
1. Apri la dashboard.
2. Vai su Database.
3. Premi "Importa rose ufficiali FIFA".
4. Attendi il log con numero giocatori e squadre.
5. La dashboard ricalcola Money Mode e Fantamondiale sui giocatori ufficiali.

Nota:
Il parsing dei nomi dal PDF FIFA è automatico. In caso di formato PDF cambiato, il log ti dirà "Parsing incompleto".
